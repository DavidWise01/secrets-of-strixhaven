# SECRETS OF STRIXHAVEN · SOS

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARENA**.

SECRETS OF STRIXHAVEN (SOS) — an ARENA Magic: The Gathering sphere: Magic's 2026 return to the mage university and its Mystical Archive, curated by basic worth. Pulls all ~271 cards LIVE from Scryfall (card data + the non-foil USD it mirrors from TCGplayer) and buckets them into worth tiers — 🩸 chase ($5+) · staples ($1–5) · role-players ($0.25–1) · bulk (&lt;$0.25) — with a set-value summary (sum / top card / chase count), search, rarity filter, foil-worth toggle, and a Scryfall thumbnail per card. The curation is the tiering + the read applied to REAL numbers (verified live: 271 cards, ~$175 non-foil set value, top Emeritus of Woe // Demonic Tutor ~$14 — a fresh set, prices still settling, 7 chase). Arcane palette (mage violet + gold leaf). Honest: prices are a live snapshot and fluctuate daily (this card moved between two checks during the build), an unlisted card shows —; card names &amp; prices are facts, images hot-linked from Scryfall's CDN (sanctioned non-commercial use); an unaffiliated fan price-reference — Magic is © Wizards of the Coast.
